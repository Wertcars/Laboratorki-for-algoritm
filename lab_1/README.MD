# Description of lab

This lab demonstrates how Python handles object binding, rebinding, mutation, function arguments, default arguments, copying semantics, and reference counting.

The program consists of six independent tasks (A–F). Each task explores a specific concept of Python’s object model and memory management using printed output, object IDs, and reference counts

# Requirement

- Python 3.11+
- Windows/Linux/macOS

# Command to Run the Program

```bash
python src\lab_1.py
```
#Description of program

### The program prints six structured sections (A–F), each focused on a specific Python memory and object behavior concept:

- Binding vs Rebinding

- Mutation vs Rebinding

- Function Argument Binding

- Default Mutable Argument Behavior

- Shallow vs Deep Copy

- Reference Counting

### Each section prints:

- Clear section headers

- Object values

- Object IDs (where relevant)

- Reference counts (where relevant)

- Separator lines between tasks

#### The output is designed to visually demonstrate how Python manages object references and memory internally.
